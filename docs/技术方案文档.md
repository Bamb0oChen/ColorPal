# ColorPal 技术方案文档

> **项目代号**：ColorPal — 你的色彩伙伴
> **版本**：v2.0
> **日期**：2026.05
> **架构**：Vue 3 + FastAPI（All-in-One）+ SQLite
> **状态**：MVP 阶段

---

## 1. 架构总览

### 1.1 系统架构

```
┌──────────────┐     HTTP     ┌──────────────────────────────────┐
│  Vue 3 前端   │ ←────────→  │        FastAPI（单进程）           │
│  (Web App)    │             │                                  │
│  + TypeScript │             │  ┌──────────┐  ┌──────────────┐  │
│  + Vite       │             │  │ AI 服务   │  │ 业务 CRUD    │  │
│  + Pinia      │             │  │ (GPT-4V)  │  │ (SQLAlchemy) │  │
│  + Axios      │             │  │ 规则评分   │  │ 任务/图鉴    │  │
└──────────────┘              │  └──────────┘  └──────┬───────┘  │
                              │                        │          │
                              │                   ┌────┴────┐     │
                              │                   │ SQLite  │     │
                              │                   └─────────┘     │
                              └──────────────────────────────────┘
```

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **最少语言** | Python + TypeScript，去掉 C# |
| **最少服务** | FastAPI 一锅端，不拆微服务 |
| **最少跳转** | 前端 → FastAPI 一次 HTTP，无跨服务转发 |
| **最少依赖** | SQLite 起步，不上 PostgreSQL |
| **Docker 仅部署** | 本地直接启动，Docker 最后再配 |

### 1.3 单次拍照数据流

```
浏览器 → POST /api/v1/photo/analyze → FastAPI
                                          ├──→ GPT-4V（主路径）
                                          │     ↑ 超时/失败 ↓
                                          ├──→ 规则评分引擎（兜底）
                                          ├──→ SQLite 存储
                                          ←── 返回分析结果
```

## 4. 数据模型（简版）

### users
```
id | nickname | pet_stage | pet_mood | pet_color
energy_current | energy_max | energy_r | energy_g | energy_b
total_photos | highest_score | streak_days | last_photo_at
```

### photos
```
id | user_id | image_base64
dominant_color | palette(JSON) | score | comment
color_category | saturation_level | brightness_level
latitude | longitude | place_name
taken_at
```

### tasks
```
id | user_id | type | title | description
target_value | reward_energy | status | assigned_at
```

## 5. API 一览

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/photo/analyze` | 上传照片 + AI 分析 |

## 7. 降级策略

| 风险 | 应对 |
|------|------|
| GPT-4V 超时或不可用 | FastAPI 自动降级到规则评分引擎，前端无感 |
| SQLite 并发写入冲突 | MVP 单用户场景不会触发 |
| 相机 API 浏览器不兼容 | 降级到 `<input type="file">` |
| 地图加载慢 | CDN 加载 Leaflet，兜底静态图 |
